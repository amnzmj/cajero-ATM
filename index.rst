Documentación del Cajero Automático
====================================

Bienvenido a la documentación del Cajero Automático desarrollado en Python con Tkinter.  
Este proyecto permite realizar operaciones bancarias básicas como ingresar con una cuenta, consultar saldo, depositar y retirar dinero.

=================
Uso del Cajero
=================

El cajero tiene la siguiente funcionalidad:

1. **Inicio de sesión**  
   - Se ingresa el número de cuenta y PIN para acceder al menú principal.
   
2. **Menú principal**  
   - Permite acceder a las siguientes operaciones:
     - Retirar dinero.
     - Consultar saldo.
     - Depositar dinero.

3. **Operaciones**  
   - Cada operación abrirá una nueva ventana donde el usuario podrá completar la transacción.
